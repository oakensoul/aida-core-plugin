# PHP Test Project

Sample PHP project for testing AIDA functionality in PHP environments.

## Structure

- `src/` - Source code
- `tests/` - Unit tests
- `composer.json` - PHP dependencies

## Purpose

Tests AIDA functionality in a typical PHP project environment:

- Composer dependency management
- PSR-4 autoloading
- PHP-specific workflows

## Usage

```bash
composer install
composer test
```
