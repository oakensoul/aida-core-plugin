# AIDA Test Project

This is a sample AIDA project for testing AIDA functionality across different environments.

## Structure

- `.claude/` - AIDA configuration
  - `skills/` - Skills and auto-triggered behaviors
  - `agents/` - Personality configurations
  - `knowledge/` - Project-specific knowledge

## Purpose

Used for testing:

- AIDA installation
- Command execution
- Memory management
- Workflow commands
- Project-specific behavior
