# Node.js Test Project

Sample Node.js project for testing AIDA functionality in Node.js environments.

## Structure

- `src/` - Source code
- `test/` - Unit tests
- `package.json` - Node.js dependencies

## Purpose

Tests AIDA functionality in a typical Node.js project environment:

- npm dependency management
- ES modules
- Node.js-specific workflows

## Usage

```bash
npm install
npm start
npm test
```
